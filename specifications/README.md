# Project Specifications

This folder contains the written hand-in materials for the Air Ticket Reservation System.

## Files

- `application_files.md`: Lists the application files and explains what each file contains.
- `use_cases_and_queries.md`: Lists the main use cases and the SQL queries executed by each use case.
- `team_contributions.md`: Summarizes who did what for the project.
- `demo_notes.md`: Provides demo credentials, setup notes, and useful test flows.

The application itself is a Flask + MySQL web app. The database schema is defined in `../sql/schema.sql`, and sample data is defined in `../sql/seed.sql`.
